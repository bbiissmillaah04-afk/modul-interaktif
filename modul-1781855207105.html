Dalam modul ini Anda akan belajar dasar-dasar pemrograman visual menggunakan bahasa pemrograman modern serta alat bantu desain antarmuka pengguna.

Materi Pembelajaran

1. Konsep Dasar Pemrograman Visual

  • Elemen GUI: tombol, teks, kotak input, canvas.
  • Event-driven programming: menangani klik, perubahan, dsb.
  • Layout dan responsivitas.

2. Alat dan Bahasa yang Digunakan

  • HTML5 + CSS3 + JavaScript untuk aplikasi web.
  • Framework populer: React, Vue, atau sederhana dengan vanilla JS.
  • Alat desain: Figma, Adobe XD untuk prototipe.

3. Praktik Membuat Aplikasi Sederhana

  1. Buat struktur HTML dasar.
  2. Styling dengan CSS (flexbox/grid).
  3. Tambahkan logika interaksi dengan JavaScript.
  4. Uji dan perbaiki responsivitas.
